# Politieke tool Render wrapper

Publieke wrapper zonder dossierdata. Render gebruikt deze repo alleen om de private repo
`jonathanvanderelst1997/politieke-tool-jonathan` read-only te clonen via een deploy key.

Vereiste Render environment variables:

- `GH_DEPLOY_KEY_B64`
- `POLITIEK_ONLINE_AUTH_USER`
- `POLITIEK_ONLINE_AUTH_PASSWORD`
- `POLITIEK_META_APP_ID`
- `POLITIEK_META_APP_SECRET`
- `POLITIEK_META_CONFIG_ID`
